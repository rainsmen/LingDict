## 🚀 使用流程

### 开发者操作

#### 方案B：自动导入（推荐）

```bash
# 1. 下载ECDICT数据
wget https://github.com/skywind3000/ECDICT/raw/master/ecdict.csv

# 2. 准备词库文件
python3 scripts/prepare_ecdict_for_assets.py ecdict.csv

# 输出: app/src/main/assets/ecdict.csv (~10MB)

# 3. 编译安装
./gradlew installDebug

# 完成！应用会自动处理导入
```

### 用户体验

```
用户首次启动应用
    ↓
显示Splash Screen
    ↓
自动检测词库

未导入？开始导入

    └─ 30-60秒完成
    ↓
导入完成
    ↓
自动跳转主界面
    ↓
后续启动秒进
```

影响因素：
- CSV文件大小
