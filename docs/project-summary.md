## 🎉 项目概览

**LingDict** 是一款基于科学记忆算法（SM-2）的智能生词学习应用，采用现代Android开发技术栈，提供沉浸式的单词学习体验。

**开发周期**：Phase 1-6 完成（Phase 7准备就绪）
**整体状态**：修复收敛中
**代码质量**：需在配置 Android SDK 后重新验证
**可发布状态**：暂不建议发布

## 🏗️ 技术架构

### 架构模式
```
┌─────────────────────────────────────┐
│         Presentation Layer          │
│   (Compose UI + ViewModels)         │
├─────────────────────────────────────┤
│          Domain Layer               │
│   (UseCases + Models + Repository)  │
├─────────────────────────────────────┤
│           Data Layer                │
│   (Room + Retrofit + DataStore)     │
└─────────────────────────────────────┘
```

### 技术栈

**UI层**：
- Jetpack Compose（声明式UI）
- Material 3（设计系统）
- Navigation Compose（导航）
- Coil（图片加载）

**业务层**：
- Kotlin Coroutines（异步）
- Flow（响应式数据流）
- Hilt（依赖注入）

**数据层**：
- Room（本地数据库）
- Retrofit（网络请求）
- DataStore（设置持久化）
- Gson（JSON解析）

**测试**：
- JUnit 4（单元测试）
- MockK（模拟框架）
- Turbine（Flow测试）
- AndroidX Test（集成测试）
